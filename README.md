Proyek ini merupakan analisis eksploratif (EDA) terhadap dataset IMDb Top 1000 Movies, yang berisi informasi seputar film dan serial dengan rating tertinggi.
Tujuan utama analisis ini adalah untuk memahami pola dan hubungan antara variabel seperti rating IMDb, pendapatan (Gross), genre, serta sutradara dan aktor yang berperan.
Analisis dilakukan menggunakan Python (Google Colab) dan divisualisasikan dengan Matplotlib, Seaborn, serta Tableau Public untuk dashboard interaktif.

Menganalisis sutradara dan dekade dengan performa terbaik

Menemukan insight menarik dari hubungan antar variabel numerik
